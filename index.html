Core Features

Everything you need to manage a modern lab

From exam lockdown to live broadcasts — LockStep gives you complete control over every device in your lab.

Secure Assessment

Exam Mode

Lock down every device in the room with a single click. Exam mode disables Alt+Tab, blocks Task Manager, restricts USB access, and clears the clipboard — ensuring a fair testing environment for every student.

  • Keyboard shortcut interception (Alt+Tab, Win key, Ctrl+Alt+Del prompt)
  • USB storage blocking and clipboard restriction
  • Tamper-proof agent — students cannot terminate the process
  • One-click activation across all lab devices simultaneously

Platform Capabilities

Built for scale, designed for simplicity

Cross-Platform Agent

Electron-based agent runs on Windows, macOS, and Linux with identical capabilities.

Role-Based Access

Superadmin, admin, and assistant roles with granular permission controls and optional 2FA.

Remote Commands

Push URLs, run scripts, restart, or shut down devices remotely — individually or in bulk.

Smart Scheduling

Schedule commands and exam sessions ahead of time. Automate lab power-on and shutdown routines.

Enterprise Security

JWT auth with refresh rotation, rate limiting, NoSQL injection protection, and security headers.

Docker & PM2 Ready

Deploy with Docker Compose or scale horizontally with PM2 cluster mode and Redis pub/sub.

Getting Started

Up and running in 3 steps

01

Deploy the Server

Run the backend with Docker or PM2. Connect to your MongoDB instance and optionally configure Redis for multi-node scaling.

02

Install the Agent

Deploy the LockStep agent on lab devices. It auto-registers with the server over WebSocket and starts reporting metrics immediately.

03

Take Control

Open the dashboard to monitor, command, and secure devices. Activate exam mode, broadcast your screen, or push commands — all in real time.
